Проверка попадания точки в область

Java FastCGI сервис для Apache, который валидирует входные данные, вычисляет факт попадания и ведёт историю запросов с миллисекундной точностью.

Геометрия области

Используемая область составлена из прямоугольника в третьей четверти, прямоугольного треугольника в четвёртой четверти и четверти круга в первой четверти. Параметр R определяет размеры всех фигур.

  • Четверть круга радиуса R/2 в первой четверти.
  • Прямоугольный треугольник с катетами R/2 в четвёртой четверти.
  • Прямоугольник размерами R × R/2 в третьей четверти.
